# 川西15天自驾攻略 · 甘孜东线深度游

家庭自驾路书，单 HTML 页面，无构建依赖。

## 路线

成都→泸定桥(途经)→中谷村(3晚)→甲根坝(3晚)→道孚(2晚)→🟢环线(丹巴→四姑娘山→成都)→成都休整(3晚)→上海

三个大本营逐级适应海拔：2500m → 3000m → 2980m。

## 功能

- 响应式：PC 960px / 手机 480px
- Leaflet 路线地图（高德瓦片，去程灰线 + 返程绿线）
- 每日行程卡（可折叠）+ 景点/酒店推荐 + 导航按钮
- 加油站提醒、高反手册、宠物方案、安全守则
- 日历表（基于出发日期）、ICS 下载、清单勾选(localStorage)
- 国道风景指南（G318/G350/S434/G548）
- 23 条小红书笔记链接
