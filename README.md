# 川西17天自驾慢游攻略 · 甘孜东线深度游

两车两家庭（4大3小+4狗）川西自驾路书，单 HTML 页面，无构建依赖。

## 路线

上海→宜昌→三峡→万州→成都→**康定**(3晚D4-D6)→**甲根坝**(3晚D7-D9)→**道孚**(2晚D10-D11)→金小路S451→小金→四姑娘山→重庆(2晚D13-D14)→武汉→上海

三大固定大本营逐级适应海拔：2500m → 3000m → 2980m。

## 功能

- 响应式：PC 960px / 手机 480px，深色模式
- 17天每日详细行程卡（可折叠）+ 景点/酒店推荐 + 导航按钮
- 三大本营设计：加油站提醒、高反手册、宠物方案、安全守则
- 动态日历表（基于出发日期）、ICS日历导出、清单勾选(localStorage)
- 国道风景指南（G318/G350/S434/G548）
- 金小路S451新景观公路攻略
- 理想L6纯电版充电指南
- 丰富的出处链接（小红书/携程等）
