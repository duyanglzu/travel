# 第一大本营修正方案：康定 2500m

## 问题背景

原方案第一大本营设在**中谷村**（D4-D6 连住3晚），日间游计划往南门关沟（轻徒步）和雅家梗红石滩。

地理分析发现：中谷村位于康定以北 30km（S434省道上），而南门关沟/雅家梗均位于康定东南方向（榆磨路）。这意味着每日出行需先折返康定再往东南，形成不必要回头路。

## 修正方案

**第一大本营改为康定城区（2500m），D4-D6 连住3晚，日间出游分别往北（S434）和往东南（榆磨路），无需折返。**

**为什么住康定而非中谷村？**
- "海拔2500跟丽江一样，不存在高反"（679赞原话）
- "唯一有星巴克、肯德基还有9.9瑞幸的地方"（679赞原话）
- "如果第一天赶往新都桥，身体疲累容易引发高反"（679赞原话）

## 关键变更点

1. 第一大本营位置：**中谷村 → 康定城区**
2. D5（北线）原"南门关沟轻徒步"→ 改为"S434北线一日游"
3. D6（东南线）原"雅家梗备选"→ 改为"榆磨路东南线一日游（含南门关沟+雅家梗+磨西）"
4. D4 新增强制途经点：天全服务区 → 二郎山隧道（初见雪山）
5. S434 详细途经点来自小红书实勘（红海子→机场彩虹公路→雅拉雪山观景台→青绕神山）
6. **备注**：如到康定后时间充裕，D4当天下午可直接先去雅家梗方向（D6空出变为休整/机动日）
