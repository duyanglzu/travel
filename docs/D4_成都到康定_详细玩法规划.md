# D4 成都→康定 详细玩法规划

> 基于小红书高赞笔记 + 携程实勘游记 + 抖音攻略视频信息整合
> 日期：08/10 · 出发地：成都 · 目的地：康定（大本营①，海拔2560m）

---

## 📊 路线总览

```
成都(08:00) → 天全服务区(10:30) → 二郎山隧道(11:30) → 兴康特大桥(12:00)
→ 泸定桥(12:15-14:15) → 榆磨路·红石滩(14:30-15:30) → 雅家埂垭口(15:45-16:15)
→ 康定城区(16:40) → 溜溜城·情歌广场(19:30-21:00)
```

**总里程**：约330km | **纯行车**：约5h | **含游玩**：约8.5h

## ⏰ 逐站详细规划

### 1️⃣ 天全服务区（10:30-11:15 | 停留45min）

|------|------|

- [ ] **"此生必驾318"路牌打卡**：停车场车道中间有大型地标，适合无人机航拍（注意来车）

- [ ] **补给采购**：氧气罐（4瓶装够到康定）、红景天、晕车贴、防寒帽、318主题车贴

**美食推荐**（选1-2样垫肚子，别吃太饱容易高反）：

- ☕ "进藏第一杯奶茶"——猫屎咖啡

### 2️⃣ 二郎山隧道（11:30 | 通过15min）

**定位**：川藏第一隧，成都平原→甘孜藏区的地理分界

| 项目 | 详情 |
|------|------|
| 📍 位置 | 雅康高速，天全→泸定之间 |
| 📏 长度 | 13.459公里 |
| ⏱ 通过时间 | 约15分钟 |
| 💰 费用 | 含在高速费中 |

**驾驶提示**：
- ⚠️ **走左道！** 右道大货车多，左道通畅（携程实勘多次强调）
- 🏔️ 过隧道后**初见雪山**——川藏线第一个震撼瞬间，车窗风景突变
- 🌡️ 过隧道后温度明显下降，体感从平原切换到高原
- 小红书@撒拉哈的马里奥 679赞原话："过隧道初见雪山"

### 4️⃣ 泸定桥（12:15-14:15 | 游玩2h含午餐）

**定位**：红色历史地标，大渡河畔的百年铁索桥

| 项目 | 详情 |
|------|------|
| 📍 地址 | 泸定县泸桥镇泸定桥广场西100米 |
| 🎫 门票 | 10元（售票处在桥东头）|
| ⏰ 开放 | 夏秋季 07:00-19:00 / 冬春季 08:00-18:00 |
| 🅿️ 停车 | 桥西头景区停车场，10元/天 |
| 📞 电话 | 0836-3128266 |

**游玩路线**（2小时含午餐）：

```
停车场(桥西) → 观音阁(俯瞰全景) → 上桥(东行) → 桥东泸定县城 → 美食街午餐 → 返回
```

**必做清单**：
- [ ] **上桥体验**：13根铁索托起103米桥面，木板缝隙可见大渡河奔腾（抓紧铁链！）
- [ ] **观音阁观景**：桥西头制高点，俯瞰整座泸定桥全景，拍大场景照
- [ ] **红军飞夺泸定桥纪念馆**（免费）：4D影院 + 22勇士墙
- [ ] **康熙御碑**：桥东头，始建于康熙四十四年

**⚠️ 安全提醒**（多篇攻略强调）：
- 穿**防滑鞋**！木板缝隙能看见河水
- 手机挂绳 + 双肩包，风大易丢东西（有游客手机"扑通"入水）
- 旺季建议**7点前到**避开旅行团，我们中午到人稍多但可接受
- 桥面摇晃正常，别走到桥两边

**午餐推荐**（桥东泸定县城美食街）：

### 5️⃣ 榆磨路·红石滩（14:30-15:30 | 边走边停1h）

**定位**：贡嘎东坡最美自驾公路，免费的高品质风景线

**路线**：泸定出发 → 往磨西镇方向上榆磨路 → 沿途红石滩观景台 → 雅家埂垭口

**榆磨路风景分层**（携程实勘描述）：
1. **深郁原始森林**段——参天冷杉，溪流潺潺
2. **红石滩**段——路旁陆续出现红石，河水从冰川奔涌而下清澈见底
3. **高山草甸**段——视野开阔，贡嘎雪山再次出现
4. **雪山溪流瀑布**段——四面环山，多条雪山溪流从高处落下

**沿途停靠点**：
- 🚗 **路边红石滩**：出磨西镇不远路旁，可下车玩水，红石上的藻类像丝绒
- 🚗 **"中国红石滩"观景台**：盘山路最高处，视野最棒——冰川雪水从贡嘎雪顶顺流而下，沿红石中间蜿蜒流淌
- 🚗 **雅家情海**：高山湖泊（海子），藏在山坡密林中，有停车场
- 🐒 **藏酋猴**：路上可能遇到，注意不要喂食

**驾驶提示**：
- 全程柏油路，路况良好
- 山路盘绕、弯急坡陡，需有一定驾驶经验
- 多个停车带与观景台，安全指示齐全
- 7月雨季注意湿滑，减速慢行

**小红书出处**：@877赞"离康定很近没人去的地方"（原话："路上几乎没人没车"）

### 6️⃣ 雅家埂垭口（15:45-16:15 | 停留30min）

**定位**：榆磨路最高点，红石滩+雪山倒影同框

| 项目 | 详情 |
|------|------|
| 📍 坐标 | 29.90028, 102.00174（泸定县燕子沟镇） |
| 📏 海拔 | 3830m（携程）/ 3800m（小红书） |
| 💰 费用 | 全程免费 |
| ⏱ 建议停留 | 20-30分钟拍照 |

**看点**：
- 🪨 **红石滩+雪山倒影**：877赞确认"几乎没人没车"，红石滩与雪山倒影湖面同框
- 🏔️ 巨石题刻"雅家埂"，侧面标注"中国高山植物园"
- 📸 最佳机位：垭口停车场，面向贡嘎方向拍红石+雪山

**⚠️ 注意**：
- 海拔3830m，短暂停留无压力，但**不要跑跳**
- 7-8月雨季山顶可能起雾，如果浓雾能见度差不要久留
- 垭口风大温度低（比山下低10℃+），穿冲锋衣下车

### 7️⃣ 康定城区·入住休整（16:40-18:30）

**定位**：情歌之城，海拔2560m（与丽江持平），川西最完善的后勤补给站

**到达后优先做**：
1. **加油站加满油！** S434北线（D5）康定到塔公全程无加油站
2. 入住酒店，休整1小时
3. **首日规则**：不洗澡、不剧烈运动、不饮酒，清淡饮食

**康定优势**（679赞原话）：
> "海拔2500跟丽江一样，不存在高反"
> "唯一有星巴克、肯德基还有9.9瑞幸的地方"
> "如果第一天赶往新都桥，身体疲累容易引发高反"

**采购清单**（康定比后续点便宜）：
- 便携氧气瓶（25元/罐，网购8元，建议买4-6瓶备用）
- 葡萄糖口服液
- 零食干粮（后续路上补给点少）

## ⚠️ 关键提醒

### 驾驶
1. **二郎山隧道走左道**——右道大货车多
2. **导航设途经点**：天全服务区 → 泸定桥 → 雅家埂垭口 → 康定（走榆磨路，不是直达高速）
3. 榆磨路弯急坡陡但路面良好，2-3年驾龄可安心通行
4. **16:00前到康定**，确保天黑前入住（榆磨路无路灯）

### 高反
1. 康定2560m与丽江持平，正常无高反
2. 雅家埂垭口3830m短暂停留OK，**不跑跳**
3. 首日三不：**不洗澡、不剧烈运动、不饮酒**
4. 天全服务区提前买氧气罐备用

### 补给
1. **康定加满油！** D5走S434全程无加油站
2. 氧气瓶康定买25元/罐，网购8元——建议成都提前网购快递到酒店
3. 康定有星巴克/瑞幸/肯德基，是后续行程最后的城市级补给

### 天气
1. 8月雨季，榆磨路可能起雾，垭口能见度注意
2. 垭口温度比山下低10℃+，带冲锋衣
3. 防晒+墨镜必备，高原紫外线强
